BDT is CipherTrust Batch Data Transformation Utility.

Bootstrap
---------

Install these tools.

- Install JDK 1.8 and set JAVA_HOME.
	It can be found at this shared location "\\10.164.8.95\ProtectDB\Software"
		OR
	Visit this official link to download "https://www.oracle.com/in/java/technologies/javase/javase8-archive-downloads.html"
- Install Maven 3.6 or above and set MAVEN_HOME.
	Visit this official link to download "https://maven.apache.org/download.cgi"

Install
-------

Clone BDT code
	git clone git@gitlabent.thalesesec.com:vacs/bdt.git
The default branch is master which has the latest code.

Build
-----

- Change directory to BDT home
	cd bdt
- Run
	mvn package
